Pengalaman Menggunakan Amazon EC2

24 April 2025 by Admin

Amazon EC2 (Elastic Compute Cloud) adalah layanan web yang menyediakan kapasitas komputasi yang aman dan dapat diubah ukurannya di cloud. Saya baru saja menyelesaikan deployment blog ini menggunakan EC2 instance.

Langkah-langkah yang saya lakukan:

  1. Membuat instance EC2 dengan Amazon Linux AMI
  2. Mengkonfigurasi security group untuk akses SSH dan HTTP
  3. Menginstall web server Apache
  4. Melakukan deployment kode blog ini

Memanfaatkan Amazon S3 untuk Backup

23 April 2025 by Admin

Amazon S3 (Simple Storage Service) adalah layanan penyimpanan objek yang menawarkan skalabilitas, ketersediaan data, keamanan, dan kinerja terkemuka di industri.

Untuk tugas ini, saya membuat bucket S3 dengan konfigurasi:

  • Nama bucket yang unik
  • Region yang sama dengan EC2 instance
  • Block all public access (akses private)

Bucket ini saya gunakan untuk menyimpan backup dari konten blog ini.
